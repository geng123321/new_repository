Markdown,文本修饰语言，用特殊符号修饰正文效果<br>

## 标题修饰符\#(\转义符加#可以输出#字符串形式，失去原有的效果) 
# 标题修饰符(一级标题) 
## (二级标题)
### (三级标题)
#### (四级标题)
##### (五级标题)

## 正文内容
  输入正文内容，但是如果需要换行，用\<br\>标签，或者空一行

## 修饰正文<br>
  *一段测试文本*

  **一段测试文本**

  ***一段测试文本***

  ~~一段测试文本~~（删除线）

## 分割线

**

---

  用\-\-\-表示分割线
## 引用效果\>表示

> 名言1
>> 名言2
>>> 3级引用
>>> 四级引用
